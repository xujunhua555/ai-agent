# AI Agent - Weekly Report System

这是一个周报定时发送助手系统，包含前后端。

## 项目结构

- `weekly-report-backend/` - Spring Boot 后端服务
- `weekly-report-frontend/` - Vue 3 前端应用

## 快速开始

### 后端

```bash
cd weekly-report-backend
mvn spring-boot:run
```

### 前端

```bash
cd weekly-report-frontend
npm install
npm run dev
```

详细文档请查看各子目录的 README.md。
